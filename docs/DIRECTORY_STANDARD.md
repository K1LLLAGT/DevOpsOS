# DevOpsOS 1.0 Directory Standard

## Top-level

- android-app/        — Android wrapper (APK, WebView shell)
- backend/            — Node.js backend
- frontend/           — Web UI
- assets/             — Shared static assets
- config/             — Config, branding, env
- plugins/            — Plugins
- marketplace/        — Marketplace metadata
- scripts/            — Utility scripts (install, patch, maintenance)
- tools/              — Dev tools, generators
- docs/               — Documentation
- server.js           — Backend entry (or wrapper)
- start-backend.sh    — Ubuntu backend launcher
- bootstrap-ubuntu.sh — Ubuntu environment bootstrap
- install.sh          — Unified installer (Termux/Ubuntu aware)
- devopsos-install.sh — Optional wrapper for install.sh
- devopsos_launcher.sh — Launcher script (Termux side)
- README.md           — Main product README
- LICENSE.md          — License
- RELEASE_NOTES.md    — Release notes
